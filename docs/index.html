WhatsApp Quick Link Native

Phone numbers on any webpage become clickable WhatsApp links — only when the number is actually on WhatsApp. Native inline, no popup, no copy-paste.

View on GitHub Privacy Policy

How it works

The extension scans every page you visit for phone-number-shaped text. For each number, it makes a single background request to wa.me to check whether the number is registered on WhatsApp. Valid numbers turn into clickable links right where they appear. Invalid numbers are left alone.

Features

Pre-validatedEvery link you see is a number that's actually on WhatsApp.
Native inlineLinks appear in place — no popup, no extra clicks.
Privacy-firstNo accounts, no analytics, no servers. Validation cached locally for 30 days.
50+ countriesBuilt-in country dial codes; infers country from the page's TLD and language.

Privacy

This extension makes no network requests except to wa.me for the number-validation check. We have no servers, no telemetry, and no analytics. See the full privacy policy.
